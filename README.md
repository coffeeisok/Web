# 易衡咨询（EpochX）团队官网

> Web 前端开发技术课程期末考核项目

## 项目简介

本项目是易衡咨询（EpochX）团队展示官网，包含公司介绍、组织架构与团队成员信息。
在原有基础上进行改造，满足 Web 前端期末考核的核心要求。

## jQuery 使用说明

### CDN 引入
```html
<script src="https://cdn.bootcdn.net/ajax/libs/jquery/3.7.1/jquery.min.js"></script>
```

### jQuery 功能清单

1. **表单验证（validation.js）**
   - 实时验证：输入框失焦时验证
   - 提交验证：提交前全量检查
   - 错误提示：红色边框 + 错误文字
   - 成功提示：绿色文字淡入显示
   - 字数统计：留言区实时计数

2. **回到顶部动画**
   - 点击页脚 Logo 平滑回到顶部
   - 使用 jQuery animate 实现

## 表单验证规则

| 字段 | 验证规则 | 错误提示 |
|-----|---------|---------|
| 姓名 | 必填，2-20字符 | 姓名必须为2-20个字符 |
| 邮箱 | 必填，有效邮箱格式 | 请输入有效的邮箱地址 |
| 电话 | 必填，11位手机号 | 请输入有效的11位手机号码 |
| 咨询类型 | 必选 | 请选择咨询类型 |
| 留言 | 选填，最多200字 | - |

## 交互功能

- **平滑滚动**：点击导航链接平滑跳转到对应区块
- **导航高亮**：滚动时自动高亮当前所在区块的导航项
- **卡片展开**：团队成员卡片可点击展开查看详情
- **滚动动画**：区块进入视口时淡入显示
- **表单验证**：实时验证 + 提交验证，带动画效果
- **回到顶部**：点击页脚 Logo 平滑回到页面顶部

## 技术栈

- HTML5 语义化标签
- CSS3（Grid、Flexbox、变量、动画）
- JavaScript ES6+
- jQuery 3.7.1
- IntersectionObserver API
- 响应式设计（移动优先）

## 开发说明

1. 所有文件使用 UTF-8 编码
2. 代码统一向右缩进（2空格或4空格）
3. 关键逻辑都有中文注释
4. 保持现有设计风格（品牌蓝 #0a3b7a，金色 #d4af37）
